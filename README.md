# Boss 打招呼生成器

一款帮助求职者快速生成 Boss 直聘打招呼消息的 AI 工具。

## 功能特点

- 📸 JD 图片/文本解析
- 🔍 技能匹配分析
- ✨ AI 生成打招呼消息
- 📄 简历上传自动生成
- 📊 用户行为统计

## 技术栈

- 前端：HTML + CSS + JavaScript
- 后端：Node.js + Express
- 数据库：MySQL
- AI：火山方舟 API
- OCR：Tesseract.js

## 本地运行

```bash
# 安装依赖
cd backend
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的配置

# 启动服务
npm start
```
